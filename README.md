# ComputerArchitecureTask3-MemoryHierarchySimulation
```
Memory Hierarchy Simulation (SSD → DRAM → Cache)
You are designing the memory subsystem of a processor datapath for Samsung
semiconductors. In real systems, instructions must travel through a hierarchy (SSD → DRAM →
Cache → CPU) without bypassing intermediate levels. This assignment focuses on modeling
the data movement across memory hierarchy.
```
